# DAA A7: Greedy Algorithms

- **prim.py**: Prim's Minimum Spanning Forest.
- **kruskal.py**: Kruskal's Minimum Spanning Tree (supports negative weights).
- **dijkstra.py**: Dijkstra's Single Source Shortest Path.
- **huffman.py**: Huffman Coding for variable-length prefix codes.

Use: `python prim.py`, `python kruskal.py`, `python dijkstra.py`, `python huffman.py`.
